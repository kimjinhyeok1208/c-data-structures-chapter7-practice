## Chapter 07. Tree (트리)
## 📘 트리 개요

- 트리는 계층적인 구조를 표현하는 비선형 자료구조이다.

### ▪ 주요 용어

- 루트(Root)

- 부모 노드 / 자식 노드

- 리프 노드(Leaf)

- 서브 트리(Sub Tree)

- 높이(Height)

---

### ✅ 문제 1. 이진 트리 생성 및 순회 구현

#### 1. 문제 개요

이진 트리를 연결 구조로 구현하고
전위 / 중위 / 후위 순회 결과를 출력한다.

- 트리 구조

 
          A
         / \
        B   C
       /   / \
      D   E   F


#### 2. 구현 내용

 이진 트리 노드 구조 정의

 노드 생성 함수 구현

 전위 / 중위 / 후위 순회 함수 구현 (재귀)

#### 3. 핵심 구현 포인트

순회는 반드시 재귀 호출을 사용

방문 순서 차이 명확히 구분

순회 방식	방문 순서

- 전위 순회	Root → Left → Right

- 중위 순회	Left → Root → Right

- 후위 순회	Left → Right → Root

### ✅ 문제 2. 이진 트리의 노드 개수와 높이 계산
#### 1. 문제 개요

주어진 이진 트리에 대해
전체 노드 개수와 트리의 높이를 계산한다.

#### 2. 구현 내용

- 노드 개수 계산 함수

- 트리 높이 계산 함수

- 재귀 구조를 이용한 계산

##### 핵심 구현 포인트

노드 개수 = 1 + 왼쪽 서브트리 + 오른쪽 서브트리

트리 높이 = max(왼쪽 높이, 오른쪽 높이) + 1

##### 시간 복잡도
연산	시간 복잡도

노드 개수	O(n)

높이 계산	O(n)

### 💡 느낀 점

트리는 재귀적 사고가 매우 중요한 자료구조임을 느꼈다.

순회 방식에 따라 출력 결과가 달라지는 점이 인상 깊었다.

BST는 단순한 구조이지만 효율적인 탐색을 가능하게 한다는 점에서 활용 가치가 높다고 생각했다.
